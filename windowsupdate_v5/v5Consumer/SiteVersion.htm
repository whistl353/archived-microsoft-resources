Release information

Thank you for participating in this pre-release of the latest version of Microsoft® Windows® Update. Your comments will help us further improve the Web site.

This latest version includes some new or improved features you will notice throughout the site. If you participated in the beta release of this website, you will notice a few additional features in this release (described at the bottom of this page).

Features in this pre-release:

  • Express and custom installation options.  We know that many customers come to Windows Update to search for only the most critical updates in order to get their computers upgraded and more secure in the shortest amount of time. We created the Express and Custom installation paths to help you quickly find the route through the site that best meets your needs. Now you can quickly get all of the high priority updates or spend more time browsing and choosing from the optional updates as well..
  • Update categorization.  You may notice that the update categories in the left navigation pane have changed a bit. These new categories enable Microsoft to better accommodate more updates over time.
  • Single installation updates.  Some updates require you to install them independently of other updates. Microsoft is working to require fewer of this type of update, but we have also changed how Windows Update offers them. If you are offered a "single installation" update, you can choose to either select it alone or select other updates applicable to your computer. By making this choice up front, you won't have to back up at the end of the process to change your selection, as was previously required.
  • Superseding updates.  Some updates replace, or supersede, previous versions of updates. For example, a service pack that includes Security Patch A would supersede Security Patch A. Windows Update automatically offers you the latest applicable update, so you don't have to choose between the latest releases and the previous versions.  
  • Update details.  We have heard from customers that they want as much information as possible about updates. We have revised our details pages to provide a fixed set of information, in an effort to make the update details more consistent.
  • Select all  Many customers have requested a one-click option to select all available updates. Although Windows Update will continue to select for you the high priority updates your computer needs, if you truly want all the available updates, you can now select them with one click when you choose Custom Install.
  • Administrator information.  Many customers may not be aware that Windows Update offers a searchable catalog of the updates we publish. Whereas the Windows Update site detects and offers only those updates that are specific to your computer(s), the Windows Update Catalog site does not, so we do not recommend it for most customers. However, advanced users who need to administer updates to other computers may want to use this feature. We have also provided information about Windows Update Services, another product that enables small- to medium-sized businesses to search for and deploy updates.
  • Hide and restore updates.  From time to time you might be offered an update that, for whatever reason, you do not want to install. You can now hide updates you don't want to see; this keeps the list of available updates from being cluttered. You can go to the Restore Hidden Updates page to retrieve the update if you change your mind later.
  • Update news.  When a critical issue comes up that Microsoft thinks you should know about, the News from Microsoft section on the Windows Update home page displays this information. It may also include helpful tips to help you keep your computer current.
  • Beta software.  Windows Update may carry beta (pre-release) software and updates to beta software from time to time. We realize that beta software is not often recommended for or wanted by all customers. If you want to be offered beta software releases when you visit Windows Update, go to the Advanced settings page and select the check box labeled "Show beta products and related updates."
  • Smart downloading.  If you lose your Internet connection while downloading, the download will pick up where you left off the next time you try to download the update. If you use Automatic Updates, Windows Update will not duplicate any updates that Automatic Updates has already downloaded for you.

Features added since Beta 1:

  • Troubleshooter search.  Our product support engineers have put together valuable troubleshooting steps for issues that customers frequently encounter while using Windows Update. We have improved our search and filtering capabilities to make it easier for you to find and use the troubleshooting content.
  • History.The update history feature is now available. It includes more details about the updates you’ve installed and any errors that may be logged in your history, as well as offering you printing options.
  • Driver details. Customers have requested more nformation about the drivers that Windows Update offers. We have provided information from Device Manager to help you learn why you are being offered a particular driver.
  • Improved messages. While our goal is that you will never see an error message, should an error occur while you use Windows Update, we will automatically search our help and support content for more information about the error and possible resolution. If information exists to help you solve your problem, you will see it when you the error message appears, and you will not have to search for information on your own.
  • Revised visual design. You will notice a new header and footer that is being developed as a standard across all of Microsoft.com. We believe this new design will be easier to use because the layout and the links have been simplified.
  • Automatic Updates settings. Many customers are not aware they can use the Automatic Updates feature to stay secure and up-to-date. The Windows Update website will remind you what your current Automatic Updates settings are, suggest the optimal settings to ensure you are getting the latest security updates, and finally provide you access to modify these settings.

If you encounter bugs or technical problems with this pre-release of Windows Update, you can alert us to those issues by following the instructions on the Help and Support page.

Thank you again for your help in this release.

- The Windows Update Team
